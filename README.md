# Tiendex

## Descripción del Proyecto

Tiendex es una aplicación desarrollada utilizando un proceso automatizado y asistido por IA para optimizar el flujo de desarrollo.

## Herramientas y Proceso de Desarrollo

### Ralph TUI

Este proyecto utiliza **ralph-tui** como herramienta principal para la orquestación del proceso de desarrollo. Ralph TUI es un sistema de gestión de tareas que permite:

- Organización estructurada de historias de usuario y tareas
- Gestión automatizada del flujo de trabajo
- Seguimiento del progreso del desarrollo
- Coordinación eficiente de múltiples componentes del proyecto

### IA y Automatización

Para la ejecución de las tareas de desarrollo, utilizamos **z.ai** con el modelo **GML-4.7**, que proporciona:

- Generación de código asistida por IA
- Análisis y resolución de problemas
- Implementación automatizada de funcionalidades
- Optimización del proceso de desarrollo

## Flujo de Trabajo

1. **Planificación**: Creación de PRD (Product Requirements Document) con historias de usuario
2. **Conversión**: Transformación de PRD a formato compatible con ralph-tui (beads o JSON)
3. **Ejecución**: Automatización de tareas mediante z.ai y el modelo GML-4.7
4. **Iteración**: Refinamiento continuo del código y funcionalidades
